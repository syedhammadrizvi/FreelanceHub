# FreelanceHub
Role-based freelance marketplace desktop app built in Java Swing (NetBeans). Features Client, Freelancer &amp; Guest dashboards, in-memory ArrayList database, live freelancer updates, real-time activity graphs, custom Graphics2D splash screen, phone/email validation &amp; role-based access control. Built under 2HM Company.
